# Portfolio Project Part 2

## Overview
This repository contains the materials for **Portfolio Project Part 2**. The submission includes the Java source code files for the application, a Word document with execution screenshots, source code, and program analysis responses, and a written comparison of the Java and C++ versions of the application with a focus on performance and security.

## Contents
This project submission includes:

- **Java source code file(s)** for the application
- **ZIP file** containing multiple `.java` files, if applicable
- **Word document** that includes:
  - screenshots of the program executing
  - program analysis responses
  - source code
- **Detailed comparison paper** discussing:
  - performance differences between the Java and C++ implementations
  - which implementation may be less vulnerable to security threats
  - reasoning to support that conclusion

## Project Requirements
The submission is intended to meet the following requirements:

1. Submit the Java source code file(s).
2. If more than one `.java` file is used, submit them together in a `.zip` file.
3. Include a Word document with:
   - screenshots of the program running
   - source code
   - responses to the program analysis prompts
4. Provide a **3–4 page detailed comparison** between the Java and C++ versions of the applications.
